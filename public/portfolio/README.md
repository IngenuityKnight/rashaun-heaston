# Portfolio Assets

Add social media portfolio files here and reference them from `components/Portfolio.tsx`.

Recommended workflow:

- Export screenshots, carousel covers, or reel thumbnails as `.webp`, `.jpg`, or `.png`.
- Keep images around 1200px wide when possible.
- Use descriptive lowercase filenames, for example `city-of-detroit-community-update.webp`.
- Reference files with a root-relative path such as `/portfolio/city-of-detroit-community-update.webp`.
- For large videos, upload them to Instagram, TikTok, YouTube, Google Drive, or another host, then add the link as the portfolio item's `href`.
